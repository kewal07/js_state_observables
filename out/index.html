JSObservables

JSObservables is a javascript only observable state library which is useful in converting JS objects into observables and has handy API to manipulate the state.

This library is not yet intended to be used on production servers but merely to expand your knowledge into observables and how reactive programming works.Click on the links in the menu to know more.
